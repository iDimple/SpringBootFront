Hello! You can ask me questions about code conventions. I'll try my best to answer you~.
Our system can answer 2 types of questions.
1) If you do not know the names of coding conventions, you can find them based on their attributes.Below are examples.
(1) What should I do if the conditional expressions are too complex?
(2) How should I name boolean variables and methods?
(3) How should I write comments?
2) You can find the properties of specific coding conventions.Below are examples.
(1) What is the benefit to declare variables in the smallest scope possible?
(2) why should I define variables in the smallest scope possible?
(3) what are the effects of using "goto"?
